PENJELASAN PROGRAM MINECRAFT-STYLE GAME

    Program ini adalah game 3D bergaya Minecraft yang dibuat menggunakan Python dengan library Ursina Engine.
    Game ini menerapkan konsep transformasi geometri 3D seperti translasi, rotasi, dan skala dalam bentuk yang interaktif dan menyenangkan.
     Konsep Transformasi Geometri yang Diterapkan
   1. Translasi (Translation)

    Setiap blok dipindahkan ke posisi berbeda menggunakan koordinat (x, y, z)
    Player bergerak dengan mentranslasikan posisinya

2. Rotasi (Rotation)

        Kamera player berputar saat mouse digerakkan
        Pandangan 360 derajat di semua arah
        Ursina menangani rotasi kamera secara otomatis

3. Skala (Scale)

        Semua blok memiliki skala standar 1x1x1
        Rumah dan pohon adalah komposisi dari banyak blok kecil
        Skala konsisten untuk tampilan seragam
4. Refleksi (Reflection)

        Tidak diterapkan secara eksplisit dalam game ini
        Bisa ditambahkan dengan membalik koordinat X, Y, atau Z

Program ini adalah implementasi sederhana namun lengkap dari game 3D bergaya Minecraft yang mendemonstrasikan konsep-konsep penting dalam grafika komputer 3D,
terutama transformasi geometri (translasi, rotasi, skala). Dengan menggunakan Ursina Engine, kita bisa fokus pada logika game tanpa harus menulis kode rendering 3D yang kompleks dari nol.
